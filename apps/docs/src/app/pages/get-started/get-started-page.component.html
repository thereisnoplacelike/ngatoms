NgAtoms

Angular UI primitives you own. Copy them into your project — no runtime dependency, full control.

Installation

Install the NgAtoms CLI globally or use it with npx.

npm install -g ngatoms

Initialize your project

Run init inside your Angular project root. This creates an ngatoms.json config and optionally copies the design tokens CSS into your project.

ngatoms init

Add a component

Use add to copy a component's source files directly into your project. You own the code — edit it freely.

ngatoms add button

Available components
